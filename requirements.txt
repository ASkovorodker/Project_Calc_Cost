# Project Calc — production-зависимости (то, что идёт в exe).
#
# Установка:
#     pip install -r requirements.txt
#
# Все версии зафиксированы через ==-пиннинг для воспроизводимой
# сборки. Менять версии и состав без явной необходимости не нужно —
# это влияет на размер и стабильность calculator.exe.
#
# Структура:
#   1. Прямые зависимости — то, что project_calc реально импортирует.
#   2. Транзитивные — подтянутые pip'ом из прямых; перечислены явно
#      для воспроизводимости (полный pin-freeze).
#
# Как пересобрать pin-freeze при обновлении:
#   1. Удалить .venv, создать заново, активировать.
#   2. pip install <прямые-зависимости-без-версий>
#   3. pip freeze > requirements.txt
#   4. Восстановить эту шапку и структуру вручную.
#
# Замечание про torch: в проекте используется CPU-сборка (torch==X+cpu).
# Это намеренно — у пользователя нет GPU, CUDA-вариант раздул бы exe
# в 3-4 раза. При обновлении проверяйте, что суффикс +cpu сохранился.
# ============================================================
